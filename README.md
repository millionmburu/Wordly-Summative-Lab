# Wordly-Summative-Lab

A Single Page Application (SPA) built using HTML, CSS, JavaScript, and Bootstrap.  
The application allows users to search for English words and view their definitions, examples, synonyms, and pronunciation using a public dictionary API.



## Features

-> Search for English words
-> Display definitions, examples, and synonyms
-> Show pronunciation when available
-> Save favorite words using local storage
-> Toggle between light and dark mode
-> Dynamic updates without page refresh
-> Error handling for invalid searches or failed API requests



## Technologies Used

-> HTML
-> CSS
-> JavaScript 
-> Bootstrap 5
-> Free Dictionary API



## API Used

This project uses the Free Dictionary API:

https://api.dictionaryapi.dev/



## How It Works

1. The user enters a word in the search bar.
2. JavaScript sends a request to the Dictionary API.
3. The response is processed and displayed on the page.
4. The application shows:
   -> Word definition
   -> Phonetics
   -> Example usage
   -> Synonyms
5. If available, pronunciation audio is displayed.
6. Users can save words to favorites, stored in the browser’s local storage.
